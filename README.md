### crud_genilson_2022
 Crud: Ajax Jquery Php DataTables e Booststrap.
 Este projeto foi feito para aplicar os conhecimentos dessas ferramentas.




### 📋 Pré-requisitos
- Boostrap.
- Jquery.
- Php 8.0.
- Datatables.
- Composer.
- Servidor IIS.
- Ou servidor apache.




### 🔧 Instalação
- Baixe o projeto do Git e atualize as dependecias com o comando:
- Cmd: composer update
